oracle 更改用户名

1.用sysdba角色账号建立连接

win+r键,输入cmd
sqlplus /nolog
conn / as sysdba;

2.查询有哪些用户

select * from user$;

3.找到需要修改的用户(user#字段是唯一标识)

select * from user$ where user# = 83;

4.修改需要更改的用户名(注意新用户名要大写)

update user$ set name = 'CXDXP' where user# = 83;
commit;

5.强制刷新

alter system checkpoint;
alter system flush shared_pool;

6.修改新用户的密码(否则无法登录)

alter user CXDXP identified by mydata;

7.测试连接

conn CXDXP/mydata@mydata;

8.关闭数据库

shutdown immediate;

9.启动数据库

startup;
